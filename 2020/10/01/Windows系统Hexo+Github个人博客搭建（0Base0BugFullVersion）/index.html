Windows系统Hexo+Github个人博客搭建（0Base0BugFullVersion）

发表于 2020-10-01 更新于 2020-10-02
本文字数： 842 阅读时长 ≈ 1 分钟
准备工作：首先需要有Git、Node.js、GitHub
1.Git下载安装教程： ps:我勾选的是默认的Vim编辑器 最后会附上Vim基础操作 https://blog.csdn.net/sanxd/article/details/82624127 https://www.bilibili.com/video/BV1Fv411q7AN?from=search&seid=16816724917143934835 2.Node.js下载安装 [https://nodejs.org/en](https://nodejs.org/en) 点击 安装时一直next 安装好后可在终端查看版本node -v 和npm -v ---- 3.GitHub 没有的自己去官网Sign up win+r cmd ctrlshiftenter(管理员身份进入) → npm install -g hexo-cli(全局安装hexo博客框架，忽略WARN) → hexo -v(验证一下) → chdir(当前所处目录) → cd /(切换到根目录下) → d: (切换到D盘) → mkdir blog(新建blog文件夹) → cd blog(进入blog文件夹) → hexo init(初始化一个博客) → dir(看一下目录下生成了哪些东西) → hexo s(然后在浏览器localhost:4000打开，里面会自动生成一个Hello World的博文) → ctrl+c(断开) → hexo new ‘我的第一篇博客文章’ → cd source/_posts(切换到该目录) → dir(看是否有这篇博文) 之后点击桌面右键选择git bash here 先cd d:/blog/source/_posts 然后vim 我的第一篇博客文章.md(进入到Vim编辑器 写博文，按esc 输入:wq回车 即可保存退出Vim) → cd ../..(退回到blog目录) → hexo clean(清理一下) → hexo g(生成一下) → hexo s(重新再启动一下，localhost:4000刷新一下即可看到新写的博文) 此时本地的搭建已完成！ 4.
